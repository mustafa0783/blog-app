# 🚀 Full Stack Blog Application

A modern full stack blog platform built using **Node.js, Express.js, EJS, and MySQL** that allows users to securely register, login, and manage blog posts with complete CRUD functionality.

Designed to demonstrate practical understanding of **backend development, database management, authentication, and dynamic server-side rendering**.

---

# ✨ Features

- 🔐 User Registration & Login Authentication  
- 🛡️ Secure Session Management  
- 📝 Create New Blog Posts  
- ✏️ Edit Existing Posts  
- 🗑️ Delete Blog Posts  
- 📖 View Individual Blog Posts  
- 🔍 Search Posts by Title  
- 📊 Personalized User Dashboard  
- 📱 Responsive & Clean UI Design  

---

# ⚙️ Installation Guide

## Clone Repository

```bash
git clone https://github.com/yourusername/blog-app.git
```

## Install Dependencies

```bash
npm install
```

## Configure Environment Variables

Create `.env` file in root directory:

```env
PORT=3000
SESSION_SECRET=your_secret_key

DB_HOST=localhost
DB_USER=root
DB_PASSWORD=
DB_NAME=blog_app
```

## Run Development Server

```bash
npm run dev
```
